# PromptWars_MainChallenge
Dynamic Academic Wellness Safe-Space

This helps students monitor and improve their mental well-being during high-stakes board exams and competitive entrance tests (e.g., NEET, JEE, CET, CAT, GATE, UPSC).
Students preparing for these milestones often face severe stress, burnout, and self-doubt. 
This tool leverages GenAl to analyze open-ended daily journaling and mood logs, uncovering hidden stress triggers and emotional patterns.
